# Data Structure Simulation Visualizer

An interactive **web-based visualization tool** to help understand and simulate fundamental **data structures** through animations and step-by-step operations.<br>
This project is designed mainly for **students and beginners** to visualize how data structures work internally rather than just learning them theoretically.
